# 7-Segment Display

This project showss how to use 74HC595 shift register to control the 7-segment display

Author: **Huy Dang**

### Component Required

- (1) x Elegoo Uno R3 
- (1 or 2) x Breadboard- (1) x 74HC595- (1) x Segment Display 
- (8) x 220 ohm resistor 
- (many) x M-M wires
